# FARAGE-MAN: Channel Chomper

A satirical Pac-Man style browser game.

Play as Nigel Farage, gobble up migrant boats in the English Channel, and avoid being caught by Count Binface.

## How to Play

- Use **Arrow keys** or **WASD** (or the on-screen buttons on mobile)
- Eat the boats for points
- Eat the glowing power boats to turn the tables and eat Count Binface
- Avoid the ghosts unless they are frightened

## Features

- Works on desktop and mobile
- High score saving
- Multiple levels
- Sound effects
- Classic side tunnels
